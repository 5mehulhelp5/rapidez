# Running.shop Vue

## Setup

- `cp .env.example .env`
- Check the database credentials
- `php artisan key:generate`
- `composer install`
- `yarn`
- `yarn run prod`
- `docker-compose up -d`
- `php artisan index:products`

## Kibana

To explore the data in ElasticSearch you can use Kibana: http://localhost:5601
